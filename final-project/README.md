
# 📑 Tổng Hợp Bài Thực Hành Lập Trình

Chào mừng bạn đến với không gian lưu trữ các bài tập thực hành và bài kiểm tra của mình. Repository này được tổ chức để theo dõi tiến độ học tập qua các giai đoạn khác nhau.

## 📂 Cấu trúc thư mục

Dựa trên các kỳ thực hành hiện có:

* **`Exams_th/`**: Các bài thực hành khởi đầu hoặc tài liệu cơ bản (đã cập nhật 3 tuần trước).
* **`Exams1_th/`**: Tập hợp các bài tập và đề kiểm tra giai đoạn 1.
* **`Exams2_th/`**: Các bài tập nâng cao và phiếu học tập mới nhất (bao gồm nội dung CSS và tài liệu hướng dẫn).

---

## 📅 Nhật ký cập nhật

| Thư mục | Nội dung chính | 
| --- | --- | --- |
| **Exams1_th** | Cập nhật các bài tập thực hành tổng hợp | 
| **Exams2_th** | Cập nhật phiếu học tập CSS - BTTH02 |
| **Exams_th** | Cấu trúc đề thi và bài mẫu ban đầu | 

---

## 🛠 Công nghệ & Kiến thức trọng tâm

* **Frontend:** Tập trung vào HTML và **CSS** (Grid, Flexbox).
* **Tài liệu:** Phiếu học tập chi tiết cho từng bài thực hành (định dạng `.docx`).
* **Quản lý:** Sử dụng Git để kiểm soát phiên bản và lưu trữ bài làm.

---

## 🚀 Cách sử dụng

1. Truy cập vào từng thư mục để xem mã nguồn hoặc tài liệu tương ứng.
2. Đối với các file phiếu học tập `.docx` trong `Exams2_th`, bạn có thể tải về để xem hướng dẫn chi tiết về bài tập CSS.
3. Các thay đổi mới nhất luôn được cập nhật ở thư mục số thứ tự cao hơn.
